Detailed Description

UIRegistry - maps effect type IDs to their visual descriptors.

Mirrors EffectFactory in structure: effects are registered by TYPE_ID string at startup (in main.cpp) and looked up by the GUI when building pedal cards.

Note
Owned by MainApplication and passed by reference to MainWindow, kept in the GUI layer. Audio classes never see this registry class.
