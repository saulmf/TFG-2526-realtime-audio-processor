Detailed Description

LevelMeter - horizontal input level bar with peak hold and clip indicator.

Visual layout (left to right):

  • Green: -60 to -12 dBFS
  • Yellow: -12 to -3 dBFS
  • Red: -3 to 0 dBFS

A thin white peak-hold marker stays at the recent maximum for approx. 1.5 s before decaying. The CLIP LED turns red on any sample >= 0 dBFS and holds for approx. 2 s.
