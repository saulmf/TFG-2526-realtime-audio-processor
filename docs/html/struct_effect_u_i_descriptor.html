Detailed Description

EffectUIDescriptor - visual identity of a single effect type.

Holds all GUI-layer metadata for one effect type. Completely decoupled from the DSP implementation.

Registered in UIRegistry keyed by the same TYPE_ID string used in EffectFactory, so the two registries stay in sync without coupling.
