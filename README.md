# Digital Guitar Pedalboard 🎸

A desktop application that processes digital audio in real time, simulating the behavior of a virtual guitar pedalboard.  
The system allows the input signal to pass through a configurable chain of processing modules, each representing a different effect or stage within the audio chain.

---

## 🧩 Overview

The application receives an audio signal from an external input device, processes it in real time, and outputs the modified signal through the audio interface.  
Each processing module can be activated, adjusted, or reordered, allowing experimentation with different sound configurations similar to a traditional pedalboard or amplifier setup.

---

## ⚙️ Features

- Real-time digital audio processing  
- Modular chain of signal effects  
- Adjustable parameters for each processing stage  
- Reorderable effects chain  
- Graphical interface for visualizing the signal flow  
- Low-latency audio input/output  

---

## 💡 Concept

The project aims to reproduce digitally the behavior of analog guitar pedals and amplifiers, providing an environment where the user can observe and understand how each transformation in the signal chain affects the final sound.
